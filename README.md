# Vass
### Configuración

## Instalar dependencias
$ npm i

### Correr en local con Hot Reload (carga instantánea de cambios).

###### Para frontEnd.
$ npm run dev

###### Para backEnd.
$ npm run json-server

###### Para backEnd de pruebas.
$ npm run json-server-test

###### Para Revisar las pruebas unitarias.
$ npm run test
